# Python CLI To-Do List Manager

A simple command-line based To-Do List application built using Python.
This project allows users to manage daily tasks efficiently through a menu-driven interface.

## Features

* Add a task
* View all tasks
* Delete a specific task
* Clear entire task list
* Save tasks to a text file
* Input validation using try-except
* Implemented using Python match-case (Python 3.10+)

## Technologies Used

* Python 3.14.2
* File Handling
* Exception Handling
* Functions
* List Data Structure
* Match-Case (Switch Alternative)

## How It Works

When the program runs, the user is presented with a menu:

* Add a task
* Show tasks
* Delete a task
* Delete whole list
* Save as text file
* Exit

The program continues running until the user selects Exit.

Tasks are stored in a list and can optionally be saved to a text file (TO_DO_.txt).

## How to Run

1. Clone the repository:
git clone -> https://github.com/Navin-Verma/Console-Based-To-Do-List-in-Python.git

2. Navigate into the folder:
cd Console-Based-To-Do-List-in-Python

3. Run the script:
 python To_Do_List.py
